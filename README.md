# Portfolio Platform - React Single Page Application

A modern, responsive portfolio showcase application built with React.

## Features

- **Project Display**: View all projects in a clean card layout
- **Add Projects**: Form to dynamically add new projects
- **Search Functionality**: Filter projects by title or description
- **Persistent Storage**: Projects saved to localStorage
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Delete Projects**: Remove individual projects
